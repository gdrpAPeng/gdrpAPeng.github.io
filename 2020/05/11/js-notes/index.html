抽象类：

event loop
js 是单线程语言，所有异步都是通过同步的方式来实现的

执行机制：
分为同步任务，异步任务
又分为宏任务，微任务
宏任务执行完再执行微任务，微任务执行完再执行宏任务

bind、call、apply

  • bind 和 call、apply 的区别

    bind 返回改变上下文的一个函数

    call、apply 改变函数的 this 上下文后直接执行函数

  • call、apply 的区别

    apply 第二个参数是个数组
